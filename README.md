This project implements a secure OTP-based authentication system for a mobile-first application using FastAPI, PostgreSQL, and JWT. Users can register and log in using their phone number with OTP verification, eliminating the need for passwords. The system ensures OTPs are time-bound and single-use, stored securely in the database, and validated before authentication. Upon successful verification, a JWT token is issued for accessing protected routes. The backend follows a clean structure with proper database models, authentication flow, and API design, ensuring scalability and real-world usability.
